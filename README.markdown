korkscript
==========

korkscript is a fork of tgemit, which attempts to strip out the TorqueScript interpreter.

The ultimate goal is to allow for easy embedding into modern C++ libraries.


Compiling
---------

Project files are generated with cmake.

	mkdir build
	cd build
	cmake ..
	make
